# Fake News Detection (Machine Learning)

This repository contains a Jupyter Notebook for detecting fake news articles using Machine Learning and NLP techniques.

## About the Notebook
The notebook covers the complete ML workflow:
- Loading and understanding the dataset
- Text preprocessing and cleaning
- Converting text data into numerical features using TF-IDF
- Training machine learning models
- Evaluating model performance

## Models Used
- Logistic Regression
- Naive Bayes

## Feature Extraction
- TF-IDF Vectorizer

## Evaluation
- Accuracy score
- Classification report
- Confusion matrix

## How to Use
Open and run the notebook `FakeNewsDetection.ipynb` cell by cell.

This project is intended for learning and academic purposes.

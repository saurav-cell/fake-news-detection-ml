# Datasets Used

The datasets used in this project are hosted externally due to GitHub file size limits.

## Fake News Dataset
Source: Kaggle  
Link: https://www.kaggle.com/datasets/bhavikjikadara/fake-news-detection

Files used:
- fake.csv
- true.csv

## Usage Instructions
1. Download both CSV files from the link above
2. Place them in the same directory as `FakeNewsDetection.ipynb`
3. Run the notebook normally
